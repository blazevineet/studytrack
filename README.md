# StudyTrack — with Gemini AI Agent (FREE)

Track DSA, Web Dev, tasks, exams + an AI agent powered by Google Gemini (free).

---

## Get Your FREE Gemini API Key

1. Go to https://aistudio.google.com
2. Sign in with your Google account
3. Click **Get API Key** → **Create API Key**
4. Copy the key (starts with `AIza...`)

Free tier: 1,500 requests/day — more than enough!

---

## Deploy to Vercel

1. Unzip and upload this folder to a new GitHub repo
2. Go to vercel.com → Sign in with GitHub → New Project → select repo → Deploy
3. Go to Settings → Environment Variables → add:
   - Name: `GEMINI_API_KEY`
   - Value: your key from aistudio.google.com
4. Click Save → Redeploy

Your app is live at https://your-app.vercel.app 🎉

---

## Install on Phone

**Android**: Open URL in Chrome → tap Install banner → home screen icon

**iPhone**: Open URL in Safari → Share → Add to Home Screen → Add
